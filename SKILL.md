# asahi - AI Standup Generator

gitコミット履歴からデイリースタンドアップレポートを生成するスキル。

## 手順

### 1. オプションの確認

ユーザーのメッセージから以下を判断する：

- **対象リポジトリ**: 指定があればそのパス、なければカレントディレクトリ
- **遡る日数**: 指定があればその日数、なければ1日
- **言語**: 「日本語」「ja」等の指定があればja、なければen

### 2. コミット履歴の取得

以下のコマンドでコミット履歴を取得する：

```bash
git log --since="<N日前の日付>" --pretty=format:"%h|%ai|%an|%s" --no-merges
```

複数リポジトリが指定された場合は、それぞれで実行する。

### 3. レポート生成

取得したコミット履歴を元に、以下のフォーマットでスタンドアップレポートを生成する。

**英語(en)の場合：**

```
**Yesterday:**
- （コミット内容を自然な英語で要約、箇条書き）

**Today:**
- （昨日の作業から推測される次のステップを提案）

**Blockers:**
- None（またはコミットから推測されるリスク）
```

**日本語(ja)の場合：**

```
**昨日やったこと:**
- （コミット内容を自然な日本語で要約、箇条書き）

**今日やること:**
- （昨日の作業から推測される次のステップを提案）

**ブロッカー:**
- 特になし（またはコミットから推測されるリスク）
```

### 4. 出力ルール

- コミットメッセージをそのまま並べるのではなく、関連する作業をグルーピングして自然な文にまとめる
- 「Today」セクションは昨日の作業の文脈から合理的に推測できる範囲で提案する
- コミットが見つからない場合は「指定期間にコミットが見つかりませんでした」と伝える
- 出力はレポート本文のみ。前置きや説明は不要
