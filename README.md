# asahi

gitコミット履歴からデイリースタンドアップレポートを自動生成する Claude Code スキル。

朝会の前に「スタンドアップ生成して」と言うだけで、昨日のコミットを元にレポートが出てくる。

## インストール

`asahi.skill` ファイルを [Releases](../../releases) からダウンロードし、Claude Code にインストールする。

または、`/install-skill` でこのリポジトリのURLを指定する。

## 使い方

Claude Code で以下のように話しかける：

```
スタンドアップ生成して
standup
昨日やったことまとめて
日報書いて
```

### オプション

会話の中で指定できる：

- **日数**: 「過去3日分のスタンドアップ」
- **言語**: 「英語で」「日本語で」
- **リポジトリ**: 「~/projects/api のスタンドアップ」

## 出力例

```
**昨日やったこと:**
- ユーザー認証にJWTトークンを実装した
- セッションリフレッシュのバグを修正した
- 認証モジュールのユニットテストを追加した

**今日やること:**
- パスワードリセットフローの実装を続ける
- オープンPRのレビューとマージ

**ブロッカー:**
- 特になし
```
